Windows 安装 vs code

下载地址：https://code.visualstudio.com

选择windows版本下载，vscode有新版本时候会自动更新，重启即可更新。

傻瓜式安装一直下一步就好了

配置 vs code

安装中文简体插件

点击左侧菜单栏最后一项管理扩展，在搜索框中输入chinese，选中结果列表第一项，点击install安装。

安装完毕后右下角会提示重启VS Code，重启之后就会显示中文了。

安装go插件

启动vscode选择插件->搜go选择Go for Visual Studio Code插件点击安装即可。

安装Go语言开发工具包

按下 Ctrl+Shift+P 键在弹出的输入框中输入 go:install ——> 选择 Go:Install/Update Tools

全选所有插件后按下回车键自动安装

由于国内的网络环境的封闭基本上都会出现安装失败的情况

解决安装失败的方法

手动从github上下载这些工具插件

  • 第一步：现在自己的 GOPATH 的 pkg/mod 目录下创建 golang.org/x 目录
  • 第二步：打开cmd cd到 GOPATH/pkg/mod/golang.org/x 目录下
  • 第三步：执行 git clone https://github.com/golang/tools.git tools 命令
  • 第四步：执行 git clone https://github.com/golang/lint.git 命令
  • 第五步：按下 Ctrl+Shift+P 再次执行 Go:Install/Update Tools 命令，在弹出的窗口全选并点击确定，这一次的安装都会SUCCESSED了
