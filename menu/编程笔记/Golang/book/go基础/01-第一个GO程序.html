1、进入GOPATH目录，新建一个目录名为hello

2、在终端cd到hello目录

3、执行命令 go mod init hello

4、在hello目录下新建一个文件名为 main.go

5、在 main.go 中写下如下代码

package main    // 声明 main 包

import "fmt"    // 导入标准库 fmt

func main(){    // main函数，程序执行的入口
    fmt.Println("Hello World!")  // 打印 Hello World!
}

6、在hello目录下执行命令 go build 就会打印出 Hello World!
